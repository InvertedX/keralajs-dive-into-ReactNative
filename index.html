What is React Native

React Native lets you build mobile apps using only JavaScript. It uses the same design as React, letting you compose a rich mobile UI from declarative components.
Existing Hybrid Solutions
  • Ionic
  • Phonegap
  • Cordova
  • Onsen UI
  • etc...

Why React Native then ?

  • No Webviews
  • Faster to Build
  • One Framework, Multiple Platforms (Kinda)
  • Hot Reloading
  • Simplified UI
  • Developer Experience

Introduction to React

Why React ?

React approaches building user interfaces differently by breaking them into components

Declarative

Fast render with Virtual DOM

Component Based architecture

Components let you split the UI into independent, reusable pieces

components are like JavaScript functions.

They accept arbitrary inputs (called “props”) and return React elements

describing what should appear on the screen.

So, whenever we want to dynamicly change the content of the web page, we modify the DOM:
The HTML DOM is always tree-structured - which is allowed by the structure of HTML document. This is cool because we can traverse trees fairly easily. Unfortunately, easily doesn’t mean quickly here.
